Install

원클릭 설치 안내

macOS

  1. Illustrator를 종료합니다.
  2. script-action-update-mac.command를 더블클릭합니다.
  3. GitHub 권한 또는 macOS 암호 요청이 나오면 승인합니다.
  4. Illustrator를 다시 실행합니다.
macOS 설치 파일 전체 세팅 파일

새 PC 전체 세팅은 full-update-mac.command를 사용합니다.

Windows

  1. Illustrator를 종료합니다.
  2. script-action-update-windows.cmd를 더블클릭합니다.
  3. 권한 확인 창이 나오면 예를 선택합니다.
  4. Illustrator를 다시 실행합니다.
Windows 설치 파일 전체 세팅 파일

새 PC 전체 세팅은 full-update-windows.cmd를 사용합니다.

전체 세팅 후 1회 확인

파일 > 스크립트 > setup을 실행하면 단위, 키보드 증감, 문자 증감, 액션 세트가 한 번에 맞춰집니다. 단축키 세트는 cjh250907을 선택합니다.

카테고리 유지

3

갱신 후 새로고침

업데이트 스크립트를 실행한 뒤 패널 목록을 새로고침합니다. 안 보이면 Illustrator를 재시작합니다.

4

자주 쓰는 스크립트 고정

정렬, 색상, 삽입처럼 반복 실행이 많은 스크립트는 패널의 즐겨찾기 영역에 묶어두면 빠릅니다.
